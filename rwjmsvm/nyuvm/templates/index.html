About

This application was created by the NYU School of Medicine and adapted for Rutegers RWJMS RWJMS. The Virtual Microscope uses the Google Maps API to display, annotate, and navigate digital slides. A detailed description of the system and its impact is available at BMC medical education: Enhanced virtual microscopy for collaborative education. Triola M, Holloway W. BMC Medical Education 2011, 11:4 (26 January 2011)

This virtual microscope allows both learners and faculty to annotate slides with community peer-review. It includes a practical exam system, and each slide can include metadata from controlled vocabularies to classify and search for slides. We have released the source code of the application as an open-source project on Google Code.
